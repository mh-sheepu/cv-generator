# CV Generator

A full-stack application to generate CVs using AI.

## Structure
- **backend/**: Node.js Express API (with AI integration)
- **frontend/**: React app for user input and CV display

## Getting Started

### Backend
1. Go to `backend` folder
2. Run `npm install`
3. Run `npm start` (default port: 5000)

### Frontend
1. Go to `frontend` folder
2. Run `npm install`
3. Run `npm start` (default port: 3000)

## Features
- Input personal details, experience, education, skills
- Generate CV using AI (OpenAI integration placeholder)
- View and copy generated CV
